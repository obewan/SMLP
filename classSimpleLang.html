Fetches a localized string for a given key from the parsed JSON file.

If variables are provided, the function replaces placeholders in the format of %variable%% in the string with their corresponding values.

Parameters
keyThe key for the localized string to fetch.
variablesOptional map of variable names and their corresponding replacement values.
Returns
Localized string corresponding to the given key. If the key does not exist, returns an empty string.
